CSS Selectors

There are 3 main ways to target elements in CSS:

  • element — targets all elements of that type e.g. p { }
  • .class — targets elements with that class e.g. .box { }
  • #id — targets one specific element e.g. #main-heading { }

This text is red because of the .red class.

This text is green because of the .green class.

CSS Box Model

Every HTML element is a box. The box has 4 layers:

  • Content — the actual text or image inside
  • Padding — space between content and border (inside)
  • Border — the outline around the element
  • Margin — space outside the border (between elements)
I have margin, border and padding applied to me!

CSS Colors

Colors in CSS can be written in 3 ways:

1. Named color: color: tomato

2. Hex code: color: #3498db

3. RGB value: color: rgb(46, 204, 113)

Background color: background-color: #f39c12

CSS Text Styling

font-size: 22px — bigger text

font-weight: bold — bold text

font-style: italic — italic text

text-align: center — centered text

text-decoration: underline

text-transform: uppercase

letter-spacing: 4px — spaced out

line-height: 2 — more space between lines. This is useful for readability especially in longer paragraphs.

HTML Entities

Some characters need special codes in HTML:

  • &copy; → © (copyright)
  • &amp; → & (ampersand)
  • &lt; → < (less than)
  • &gt; → > (greater than)
  • &hearts; → ♥ (heart)
  • &nbsp; → non-breaking space (invisible but useful)

  • Inline CSS — written directly on the element using style=""

  • Internal CSS — written inside a <style> tag in the <head>
